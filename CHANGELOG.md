# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-20

### Added
- Official v1.0.0 release.
- Stable version with comprehensive footer, branding, and performance refinements.

## [0.1.4 (Beta)] - 2025-12-20

### Added
- Comprehensive footer with version, README/CHANGELOG links (via md-viewer), copyright, and hits.sh view counter.
- Hits.sh badge integration in README for view tracking.

### Changed
- Updated README features section to document all functionality from previous releases.

## [0.1.3 (Beta)] - 2025-12-19

### Changed
- Aligned branding with the PDF Splitter app, adopting the premium title gradient and typography.
- Integrated the app icon directly into the main header `<h1>` for better branding.

## [0.1.2 (Beta)] - 2025-12-19

### Added
- Embedded a high-resolution SVG favicon depicting a PDF document with reorder icons.

## [0.1.1 (Beta)] - 2025-12-19

### Changed
- Updated `README.md` with status badges for version, status, demo, license, and semver.

## [0.1.0 (Beta)] - 2025-12-19

### Added
- Upload progress bar for better visual feedback during PDF processing.
- Filename customization: reordered files now use the `_reordered` suffix.

## [0.0.1 (Beta)] - 2025-12-19

### Added
- Initial release of PDF Reorder.
- Support for single page selection and page ranges.
- Drag-and-drop file upload.
- Client-side PDF manipulation using `pdf-lib`.
- Premium responsive UI.
